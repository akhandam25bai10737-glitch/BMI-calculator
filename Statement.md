Problem Statement: Individuals require a simple, offline tool to instantly check their health status based on weight and height; this project solves that by creating a reliable, easy-to-use BMI calculator.

Scope of the Project: The scope is limited to a single Python script running in the console, focusing on data input, the core mathematical calculation, and providing a categorized output.

Target Users: The primary target users are beginner Python students learning fundamental programming concepts and any individual needing a quick, accurate metric using standard metric units.

High-Level Features: The core features include input validation to handle non-numeric entries, accurate BMI calculation, and immediate display of a corresponding health classification.
